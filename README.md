# Real-Time Tracker App

A real-time location tracking application developed using EJS for dynamic rendering, Express JS, Node JS, and JavaScript. This app allows users to track their location in real time.

## Description

This Real-Time Tracker App provides users with the ability to view their current location on a map and track their movements in real-time.It uses server-side rendering with EJS, 
and the backend is powered by Express JS and Node JS. The frontend is built with plain JavaScript to handle location tracking and updates.

## Features

- Real-time location tracking.
- Dynamic rendering of location data using EJS.
- Responsive and interactive map interface.
- Simple and user-friendly design.

## Technologies Used

- **EJS**: For dynamic content rendering.
- **Express JS**: For creating the backend server.
- **Node JS**: For running server-side code.
- **JavaScript**: For handling real-time location updates and frontend logic.
